### 个人项目脚手架
#### 通过download-git-repo来下载远程模板,支持 GitHub、 GitLab 和 Bitbucket 等
#### 提供四种命令
- 添加模板 xn-tem-add
- 删除模板 xn-tem-delete
- 查看模板 xn-tem-list
- 初始化模板 xn-tem-init template-name file-name
